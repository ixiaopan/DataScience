# Automation

- MR Notification
- Frontend Weekly

## MR Notification

### Dev

```
npm run dev:gitlab
```

### Build

```
npm run build

# 每日17点 MR 处理提醒
npm run start:gitlab:alert

# webhook
npm run start:gitlab:hook
```

## Frontend Weekly

### Dev

```
npm run dev:weekly
```

### Build

```
npm run build

# 每周一11点 发布前端最新技术
npm run start:weekly
```
